TicTacToeGame
=============
###Requirements###
Java

###Eclipse setup###
1. Run ./gradlew eclipse.
2. In eclipse open File->Import.
3. Expand "General" and choose Existing projects into Workspace.
4. Keep "Select root directory" selected and browse for the project.
5. Press "Finish".
